回应需求与有效供给

嘉定区公共文化信息服务平台“文化嘉定云”建设的思考

嘉定区文化广播影视管理局


各位专家、来宾：


我本次交流的题目是《回应需求与有效供给——嘉定区公共文化信息服务平台“文化嘉定云”建设的思考》


一、文化嘉定云建设与探索路径


有别于一般政府信息化信息“一次建设多年维护”的常规投入模式，文化嘉定云在三年的运作过程中实施了三期“新建项目”，共进行4次较大规模的改版。


（一）文化嘉定云V1.0：数字资源的汇聚与整合


文化嘉定云V1.0定位于“集展现、服务、交流等多种服务功能为一体，涵盖群众文化、公共图书、文博展览等各项公共文化数字服务”的综合性公共文化数字服务平台，设计了多达15项功能的版块集中彰显“嘉定地方文化特色”，并以“看演出、逛展馆、读报刊”这种平民式话语体系，引导市民在开放的网络空间中直接获取传统公共文化服务。网站虽然分成15个版块，但其服务方式主要有以下4种：


（1）利用互联网，延伸图书馆资源数据库使用范围。商业数据库查询服务本来是嘉定区图书馆的固有功能，但其服务范围局限于区图书馆IP范围内，其传统服务方式一方面知晓度不高，另一方面服务能级有限。该服务通过将嘉定区图书馆购买的商业数据库与文化嘉定云对接，大幅提升了区图书馆服务能级，市民凭读者证号便可在家中进行查询。


从服务效果看，这种数字化服务形式是成本较低而效能较高的，它充分利用了原有的数字资源，大大方便了嘉定市民特别是学者、学生对于专业数据库的使用，深受欢迎。


（2）整合形成嘉定区公共文化数字资源库。这一部分包含了“文化地图、看电视、听讲座、逛展馆、看展览、观演出、听广播”7个版块，占了15个版块中的近一半。事实上，该类服务形式也是整个文化嘉定云1.0版建设重点。它将政府主导下各类公共文化资源进行数字化制作，包括嘉定区广播电视台的音视频资源，嘉定区图书馆、文化馆及各社区文化活动中心的演出、讲座、展览的数字资源，嘉定博物馆、陆俨少艺术院等场馆及藏品的三维建模展示。


这一版块最初是希望通过公共数字文化的提供，弥补公共文化服务总量不足、服务不均等问题。从管理的角度看，整合全区公共文化数字资源对于地方文化的收集、整理、保护、研究还是有一定积极意义的，但是对于市民而言，数字形式的文化服务因体验感不足而缺乏吸引力，对于公共文化服务本身的推动作用较小。


（3）搭建文化信息发布平台，拓展市民文化信息获得渠道。利用互联网发布区域内文化活动信息，看展览、观演出、看电影、听讲座，方便市民获取相关信息。


这一版块的功能看似简单，却对于嘉定区整个公共文化信息服务方式产生了划时代的意义。一方面，当时政府信息服务几乎清一色的以新闻信息为主，而该网站则第一次采用了“预告”的信息服务，将公共文化服务流程前置化；另一方面，构建了较完善的覆盖全区的公共文化信息上报发布的体系和流程，形成了较全面和权威的文化主题的独立媒体，弥补了市民对于文化信息获得渠道的诉求，同时，对基层公共文化的管理也产生了积极作用。


（4）构建同城文化交流网络平台，引导市民自我服务。搭建一个共享、赏析、评价的数字文化社区，市民可以在文化互动区上传并管理自己拍摄、创作的摄影、书画、音乐、歌舞、戏曲等数字资源。


值得一提的是，文化嘉定云V1.0从项目规划到正式上线经历了2012到2014年近两年时间，而这段时间恰是中国信息化发展最迅速的两年，其中对文化嘉定云最具影响力的事件主要有两个：一是智能手机和3G、4G网络技术的成熟和普及，智能手机迎来换代高峰；二是微信用户大幅增加，P2P式交互方式变得极为便捷。可以说，文化嘉定云V1.0在建设规划的两年期间，社会信息化发展水平有了相当大的改变，诸如“同城文化互动”这类有趣的设计，在建成后就已经相对落后于市场的步伐。


因此，2014年12月，我们抓紧上马了文化嘉定云二期，以社会发展、技术进步、市民需求为依据，以问题为导向，对文化嘉定云的服务进行完善。


（二）文化嘉定云V2.0－V2.5：信息服务的建设与规范


经过一期项目的实践，我们意识到，“先设计再建设”的政府项目常规建设方式并不适合信息化项目，因为信息技术更新换代非常迅速，过长的规划建设期会导致信息化产品落后于市场，从而降低了产品使用价值和服务效能。因此，二期项目在招投标时便将“规划与建设同步、运行与调整同步”的运作方式纳入考虑因素，运作20个月期间共经历了2个大版本和37个小版本的改动，在建设中寻求更加适应基层政府的公共文化信息服务规范。


文化嘉定云从一期到二期是一个“从加法到减法”的过程，它以用户使用热度为准绳，对服务内容进行有目标的挑选和加强，服务版块从14个缩减到5个，但服务效能却成倍增长。其变化主要体现在：


（1）服务角色从生产者到提供者。在一期项目中，政府主要重心在于自己承担公共数字文化产品的直接生产；而在二期项目，政府更多关注于公共文化信息服务的内容、方式、消费边界、监督评估等规则的制定，退居幕后搭建文化服务部门与市民间的桥梁。


（2）服务目的从绝对均衡到机会均等。文化嘉定云1.0版本试图以公共产品数字转化的方式，达到公共服务“绝对均衡”，然而效果并不理想。而文化嘉定云2.0版本则转变思路，力促全区公共文化服务借助该平台实现公共服务的“机会均等与权利平等”，收获了较好的社会效益。


（3）服务模式从配送式供给到回应式供给。文化嘉定云一期项目毫无选择地罗列了嘉定区公共文化的所有服务项目与产品，形成了公共数字文化的“覆盖式供给”，这种服务模式虽然更有规模、更具体量，但毕竟带着自上而下的“配送”味，百姓并不买帐。而二期项目则充分站在市民的角度，以“回应需求”的方式实现公共文化信息服务的有效供给。


（4）服务内容从数字文化到信息服务。文化嘉定云在内容探索中，逐渐将重心从数字文化的服务转向公共文化的信息服务上，公共数字文化服务的版块以合并整合为主，而公共文化的信息服务则不断扩展、优化，实现了文化活动的网上预约、在线选座，文化场馆网上预订，同时还衍生出文化团队网上登记、个人用户实名认证等信息管理版块，促使我区公共文化同步进入规范有序的标准化管理中。


总而言之，在文化嘉定云二期项目中，我区较成功地找了建立公共文化信息服务综合平台的方向与路径，到2016年10月底，全网注册用户超过6万人，日PV访问量达到34万，增长量分别是一期同期数据的30倍和300倍。


（三）文化嘉定云V3.0：社会化与自服务的探索


文化嘉定云三期项目计划从2016年底正式上线，目前已经进入测试阶段。在确定了公共文化信息综合服务平台的基本形态后，三期开始往注重“个性化服务”的方向发展。


文化嘉定云V3.0建设内容依然以公共文化综合信息服务为重心。除了对“文化活动”、“文化场馆”等热门版块进一步优化外，探索性新增两个版块：“文化众筹”和“文化社团”。这两大版块的诞生不仅是嘉定区公共文化服务走向社会化的重要途径，也是为了解决文化嘉定云继续发展的重大障碍：服务总量不足。


以文化嘉定云最热门的文化活动版块为例，2016年上半年，网站平均每月发布文化活动信息230场，其中订票类193场，而其中70%以上的活动入场券是在一小时内订光的。这个数据当然反映了市民对于网站的关注和认同，但另一方面也意味着，更多时间中，网站可提供的优质信息服务是极少的，这造成了文化嘉定云健康发展的最大障碍。


从根源上说，公共文化信息服务总量不足是以上困境产生的原因。然而事实上，从2014年到2016年三年间，文化嘉定云的信息服务已经覆盖到嘉定区100%的区级公共文化活动、87%的镇级公共文化活动、部分村居级公共文化活动，甚至包括了文化明办等政府其它委办局对市民提供的文化服务。可以说，仅靠政府提供的公共文化服务已经无法满足市民需求。在此背景下，文化嘉定云V3.0尝试以文化众筹和文化社团两个版块，为文化类企业、非政府组织、群文团队、甚至个人的社会力量参建公共文化搭建平台，以此推动全区公共文化服务能力的提量升级。


综上所述，在跨越三年的公共文化数字化建设探索中，“文化嘉定云”经历了三个转变：从政府单向提供服务到引导群众自主服务转型；从数字文化需求机制的分享者向供需平衡的协调者转型；从追求规模的数字文化资源库建设到关注个性的文化信息服务平台建设转型。


下面，我想就文化嘉定云在探索中形成的一些经验与规范与大家进行分享。


二、文化嘉定云服务标准与规范


标准化是我区构建现代公共文化服务体系的重要战略要素之一。文化嘉定云项目通过三年的实践形成了符合嘉定区情的公共文化信息服务标准，并于2015年4月通过ISO9001标准化质量管理体系认证。下面我将从重点从内容设置、服务渠道设置、黑名单制度等几个方面，与大家分享文化嘉定云在探索中形成的一些经验与规范。


（一）内容设置


文化嘉定云以自身多年探索的经验和教训，形成了包含6项服务功能的公共文化信息服务平台基本服务标准，即：


1、信息发布服务，主要包括文化活动、文化众筹、社团招募等信息的预告与预约。


2、资源管理服务，主要包括群文团队、文化类社会组织以及各级公共文化场馆设施的管理。


3、特色数字资源服务，包括数字书刊文献、文化节目回顾、数字展馆展现等。


4、文化交流平台服务，包括文化团体管理与宣传，自媒体活动策划等。


5、文化信用体系功能，包括黑名单机制和积分机制等。


6、数据分析管理服务，包括活动、场馆、团队、实名认证等数据的收集与分析，为公共文化系统各级管理者提供决策依据。


（二）服务渠道


为了适应不同类型、不同年龄段的用户，力图达到公共文化最大程度的“广覆盖”，文化嘉定云服务提供渠道分为电脑网页、手机APP（含IOS、ANDOID系统）、终端机、微信公众号四种渠道。其中，电脑网页、手机APP和微信公众号的服务都比较常见，而终端机则是文化嘉定云区别于其它文化信息类平台较特殊的服务渠道。


终端机布点于嘉定区5个区级文化设施和13个街镇社区文化活动中心，每个设施在显要位置设置1-2台终端机，主要服务人群是社区老年人，方便其在不能靠自己或亲友预订文化活动的情况下，就近在公共文化场所，获得工作人员或志愿者的订票、取票服务。同时，它也为一些目的并不明确的到馆市民提供了解近期该场馆活动的信息渠道。


（三）黑名单制度


文化嘉定云黑名单制度是针对网站个人及团队用户在公共资源使用过程中的不良行为设置的惩治机制，包括：


（1）文化活动未入场票数达到5张（含5张），即冻结账号文化活动预订功能。


（2）文化场馆未入场场次达到3次（含3次），即冻结账号文化场馆预订功能。


（3）因故不能到场，可提前一天对活动或场馆进行网上退订。


（4）是否入场以现场验票为准，活动开始30分钟后验票程序自行关闭。


（5）黑名单账号在一个自然年结束时自动释放。


黑名单制度是商业网站不可能建立的规则，它让政府立于公共管理者的正确位置，建立起有益的社会诚信体系，对于公共文化活动秩序的建立产生了积极意义，引导市民积极参与公共文化并维护良性文化氛围。


三、文化嘉定云的不足和发展方向


（一）不足


文化嘉定云作为一个发展中的基层公共文化信息服务平台，具有一定限制和不足。


1、容易形成各地域公共文化服务水平的“新剪刀差”。


进入信息技术革命时代，文化领域同样被互联网所影响乃至重构，倒逼传统公共文化服务模式加紧拥抱互联网。一些群众基础较好、服务人员年轻化的地域，可以找到更高效、更优质的服务渠道；而在老龄化严重、无法顺利转型的地域，公共文化服务行政成本越来越高，服务能力反而下降。这种“新剪刀差”在城、乡对比之间更加明显，原因主要有：


（1）受社会发展水平影响，农村地区互联网普及、智能手机使用、wifi覆盖等信息化基础条件远不如城市地区。


（2）互联网式服务的重点在于激发用户主动获取，但农村地区居民文化需求本身低于城市地区。


（3）相较于人口较为密集的城市地区，农村地区在发展信息服务物理网点方面比较困难，且面临着更严重的技术与服务方面的缺陷。


因此，在公共文化走入信息服务的过程中，公共文化服务的“均衡性”要求将受到挑战。


2、运行模式仍以政府为主导，难以形成大文化服务格局。


文化嘉定云的建设以政府信息化项目为支持，核心内容由政府主导下事业单位运作，而受安全要求、技术力量等因素影响，社会参与的限制性仍然存在，难以形成开阔的运作机制。在文化嘉定云建设初期，由于可配置的资源有限且决策较简单，政府主导的确可以产生集中力量办大事、办快事的效果。但是，在市民文化需求日益复杂的情况下，政府主导的运行模式容易带来一些隐患，主要表现在以下三点：


一是受政府部门边界影响，服务资源难以扩展。目前文化嘉定云作为嘉定区文广局主办的项目，其服务以文广系统内部资源为主，对其它政府委办局难以造成影响。在相对独立的部门利益驱使下，同级部门为了达到类似的服务效果，很可能建立同类型的“教育云”、“体育云”等信息平台，造成低水平的资源配置重复和服务能力过剩。


二是容易形成过度服务，导致市场的相对萎缩。一面是对于基本公共服务边界规定不够清晰，一面是“服务型政府”的转型推动，很容易形成过度服务，而对文化的市场力量与民间资本的成熟造成影响。


三是政府僵化的财政和立项流程，对于信息化项目服务稳定性造成制约。政府项目规划建设过程冗长、流程繁复，且“重建设轻维护”，运维资金比较缺乏。此外，政府信息化项目自身运作成本低、生存能力强，没有市场环境中更新、创新的紧迫感，在信息技术飞速发展的现状下，网站设计与功能容易落后于市场，使得服务质量的稳定性受到挑战。


3、数据分析反馈结果进入决策系统依据不足。


目前，政府相关部门对于数字化管理方式、数据分析、信息部门的作用重视仍然不够，信息平台所收集到的市民生活习惯、文化选择等数据无法进入决策系统。除了数据分析功能尚不完善外，主要原因主要有两点：


一是管理者思想具有局限性，将信息化简单地理解为由计算机取代传统手工，对于信息部门的人员设置也缺乏综合素质的考量，并不重视信息系统带来的数据结果或数据分析。


二是传统服务理念尚未彻底清除。一些文化单位的管理者担心以数据分析为依据的决策方式会引发利益重新分配，只好选择无视来自民意的数据。


（二）发展方向


根据上述不足，未来我们将以“文化嘉定云”项目建设为核心，进一步完善公共文化服务数字化建设：


1、转变观念，进一步树立服务型政府理念。


改变政绩观，确立公民本位的服务型政府工作理念，并在制度设计、行为方式等方面进行相应的改变，使公共服务由人治模式向法制模式下的长效机制转型。


2、完善政策，建立标准化引领下的公共文化信息服务长效机制。


硬件上，将利用相对均衡的基层公共文化设施网络，进行公共文化设施wifi全覆盖等信息工程建设；制度上，将进一步明确公共文化信息服务的长期战略，建设“信息+服务”的人才梯队。此外，将努力完善科学严密的考核体系，建立以数据支持决策的公共文化决策体制。


3、制度转型，建立“新”型公共文化服务模式。


我们还将尝试建立市民“自服务模式”，政府退居“服务者”的角色，通过营造网络文化社区氛围，让市民、文化团队以及相关社会力量成为公共文化服务与管理的主角。
