  • : 民盟上海市委员会

  • : 柯力士信息安全
  • : 上海市协创工业智能装备技术研究院、无羽伦比羽毛球队

2016年3月17日下午，民盟上海市社会服务部部长王子昂宣布:2016，“柯力士”杯民盟上海市第三届羽毛球联赛正式启动，并预祝大赛圆满成功！



（中:王子昂部长，左:柯力士CEO 周黎，右:柯力士创始人 颜明华）

同时宣布，由柯力士捐赠的大型模型飞机TREX700（颜明华亲手装配），在线拍卖启动（拍卖金额将全部用作本届联赛奖金）

王子昂部长重要讲话:
      回首二〇一五年羽毛球大赛，风云际会、群贤毕至。上海民盟有十个基层组织参加了团体比赛、来自四十多个基层组织的近二百名盟友参加了个人比赛，协助参与赛事的志愿者和拉拉队员有一百七十余人、各种赛事相关的微信群和微信公众号覆盖成员高达一千余人！
      二〇一五年大赛所涌现的羽球精神、竞技精神和奉献精神，与上海的都市风貌、与民盟的传统风骨交相辉映、相得益彰、同舟共进。
      二〇一六年再聚首，喜逢上海民盟七十庆典，我们会重谱羽球乐章、再创赛事辉煌。激战五月、势不可挡！
