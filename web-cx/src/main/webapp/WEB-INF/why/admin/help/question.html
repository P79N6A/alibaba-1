封面上传失败怎么办?
  • 1、上传图片是否750*500像素以上；
  • 2、需要上传2M以下的照片；
  • 3、请使用谷歌浏览器。
活动上传中没有对应的热区位置怎么办？
  • 请及时联系文化云进行添加，联系电话：4000182346。
活动主题是什么？
  • 1、关于活动介绍的热词或者宣传语；
  • 2、最多7个字符。
为什么我的日期一直选择不成功？
  • 只能选择当日级以后的日期。
如何添加地图坐标？
  • 输入地址后，鼠标变成“小手掌”时，即可点击，抓取坐标。
如何选择票务类型？
  • 1、选择“不可预订”，代表本活动不需要预约，可直接前往，前台将显示“直接前往”的字样；
  • 2、选择“自由入座”，填写每场可预订票数即可，前台将显示“在线预订”的字样；
  • 3、选择“在线选座”，添加座位模板即可。前台将显示“在线预订”的字样。
