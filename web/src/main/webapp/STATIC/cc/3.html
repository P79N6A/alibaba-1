建设数字文化广场提升基层公共文化服务效能

——山东省东营市数字文化广场建设情况汇报

山东省东营市文化广电新闻出版局


一、主要设施功能


东营市文化广场体现数字化功能的设施主要有三类：一是无线wifi信号发射设备，有效辐射范围为半径120米，流量达到100兆；二是LED屏，同时安装高清机顶盒；三是高清摄像头。


实现的主要功能主要有三项：一是互联网功能，群众在文化广场上可利用手机、ipad免费上网，登陆“文化东营云”平台，观看各类电视节目、阅读数字书刊、浏览推送APP文化服务信息，利用市和县区公共文化数字服务平台接受文化辅导培训；二是电视收看和视频播放功能，群众可在广场上同步收看高清电视节目、进行电视视屏节目点播、5天内电视节目回放、自主播放宣传党建文化等视屏资源；三是文化远程辅导培训功能，通过高清摄像头和LED屏回传高清视频信号技术，实现市、县区文化馆和基层文化广场“一对一”或“一对二”或“一对N”文化辅导远程培训，文化馆的辅导培训教师在馆内就可对基层文化广场的群众进行同步文化辅导培训。


二、主要做法


（一）统筹规划，分步实施。2015年东营市全年全市共建成100个乡村剧场。2016年，东营市委、市政府确定将再建设100个乡村剧场和中心城建设100个群众文化广场纳入年度民生实事。在部署乡村剧场和群众性文化广场建设任务时，把实现免费wifi广场全覆盖作为一项主要建设标准。按照“试点-推广-普及”三步走战略，2016年全市完成200个数字文化广场建设任务，到2018年基本实现全市文化广场数字化建设任务，把群众性文化广场建成集群众休闲、宣传文化、党员教育、科学普及、普法教育、体育健身和民风展示等于一体的基层综合性文化服务场所。按照结构合理、实用节约、彰显特色的原则，根据广场所处位置、辐射范围、人口规模、群众文化工作基础以及经济实力等综合因素，确定建设位置，确保场所建成后真正发挥作用。


（二）制定标准，完善功能。东营市数字文化广场建设的基本标准，一是文化广场面积不少于1000平方米，配套建设阅报栏(或电子阅报屏)、体育健身设施和灯光设备;二是农村文化广场剧场舞台不少于100平方米，配备音响、播放器、舞台灯光等基本设备；三是在文化广场上开通无线网络，实现无线网络广场覆盖，条件较好的文化广场可同时安装LED屏、高清机顶盒、高清摄像头等设施；四是在广场上设立一面用于展示当地历史文化、风情风貌和刊登公益广告等内容的文化墙(廊)等。其他建设标准还有配备文化器材、培养文化活动队伍、设立文化管理员、培育文化品牌项目、建立管理运行机制等，实现“活动有队伍、演戏有场地、健身有设施、上网能免费、管理有人员”的目标。


(三)政府主导，社会参与。数字文化广场建设是政府主导的一项公益性文化惠民工程。东营市委、市政府高度重视乡村剧场、群众文化广场和数字文化广场的建设工作，2016年将其纳入年度民生实事，市政府所在地的34处群众文化广场和31处电子阅报屏建设所需830万资金全部由市财政承担，并按照每建成一处乡村剧场、数字文化广场市财政给予3万元补助的标准，对县区进行奖励。同时，积极鼓励引导社会力量参与。东营市广电网络公司积极参与数字文化广场建设，投入近百万元进行项目研发，并对具备条件的全市所有文化广场免费安装数字设施设备，目前已投入1000万元完成了152个广场的数字设备安装调试。数字化设施正常运行后，县区政府以政府购买服务的方式分期支付无线wifi使用费用，并采取“委托管理”的方式将广场的数字化设施设备的日常运行管理维护委托给该公司负责。目前，东营区已与该公司签订委托管理协议，确定自2017年开始5年内每年支付该公司68.2万元wifi使用管理费，5年之后所需费用根据实际商定解决。


（四）压实责任，强化督查。一是明确建设主体。明确县区是群众文化广场、乡村剧场、数字文化广场建设的责任主体，按照市里的统一部署，把该项工作列入重要议事日程，细化方案，落实责任，精心组织，统筹安排，确保建设工作扎实推进。同时，发挥协调机制作用，宣传、文化、党建、体育、卫生、科技、司法、共青团、妇联等部门单位协作配合，资源整合，共建共享，确保乡村剧场和数字文化广场充分发挥作用。二是强化督查考核。把乡村剧场、群众文化广场、数字文化广场建设纳入各县区年度目标责任制考核和公共文化服务体系建设考核。建立督导检查和责任追究制度，确保建设工作责任到位、投入到位、措施到位。


三、主要启示


东营市数字文化广场建设符合中央构建现代公共文化服务体系的要求，符合现代公共文化服务体系数字化建设的方向，是落实“互联网+公共文化”战略的创新性举措。


（一）数字文化广场为文化惠民服务提供了一种新模式。数字文化广场实现了“线下”与“线上”服务的有效融合，通过无线网络，将广场LED屏幕终端与各级文化馆高清视频信号互联互通，文化馆可对多个文化广场开展远程辅导培训服务，既节约了辅导培训成本，又提升了辅导培训效果，公共文化服务实现了从“体制内循环”向“社会大循环”的转变。


（二）数字文化广场为基层公共文化服务体系建设提供了一条新路子。群众文化广场、乡村剧场、数字文化广场建成后，群众主动走出家门，观看文化惠民演出，广场成了群众文化生活的好场所；群众自发参加各种休闲娱乐活动，经常接触交流，人际关系更加和谐，广场成了群众沟通交流的新桥梁；上级部门组织的下乡进社区文艺演出、电影放映、文化科技卫生法律宣讲展览等活动，以及庄户剧团、社会文艺团体“群众演给群众看”的节目，传播真善美、弘扬正能量、倡导新风尚，广场成了精神文明建设的大课堂；村委会、居委会等基层组织抓住群众在广场活动的机会，及时宣传党和国家的政策、法律，通报村务、社区管理服务事项，广场成了群众民主管理的主阵地。这种服务方式由室内向户外延伸，把文化广场打造成了一个集宣传文化、党员教育、科学普及、普法教育、体育健身等服务于一体的综合性文化服务场所，有效提升了基层公共文化设施的服务效能。


（三）数字文化广场为文化主管部门监督管理提供了一种新方式。按照创建国家示范区的标准要求，东营市每年要组织近百家专业、社会文艺院团完成10000场次的下乡进社区文艺演出任务。传统的监督管理方式，很难实现对数量庞大的文艺院团和演出节目的有效监管。乡村剧场、文化广场，特别是数字文化广场建成后，演出团体通过免费wifi登录“文化东营云”数字平台，可现场发送演出视频、图片，文化管理部门可通过数字文化广场上的高清摄像头及时调取文艺演出现场情况，群众可随时通过手机对演出节目作出评价，这极大地提高了文化主管部门的监管效率，健全完善了公共文化服务监督管理和评价机制。
